# so stupid search tool

# install

* Download the file
* Enter the directory contains file with name of "Cargo.toml"
* Run command as blow:

    > cargo build --release

* Then you will get the executable file ./target/release/sfind

# Usage
  ```
  sfind search-string start-directory
  ```

# Example

 ```
 sfind walk .
 ```
